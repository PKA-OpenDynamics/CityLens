# Changelog

All notable changes to CityLens App will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.2.0] - 2025-01-XX

### Added
- Initial release of CityLens mobile application
- Authentication system (Login, Register, Forgot Password)
- Explore screen with weather, AQI, and traffic information
- Map screen with location services
- Report screen for citizen reporting
- Profile screen with user information management
- Environment detail screen with weather and air quality data
- AI Assistant integration
- Navigation system with tab navigation
- Suggestion topics section (Địa điểm, Dịch vụ, Nhà hàng, etc.)

### Features
- Real-time weather data integration
- Air quality monitoring
- Traffic information display
- User profile management
- Report creation and viewing
- Location-based services
- Responsive UI/UX design

### Technical
- Built with React Native and Expo
- TypeScript support
- React Navigation for routing
- Expo Linear Gradient for UI
- React Native Maps for map functionality
